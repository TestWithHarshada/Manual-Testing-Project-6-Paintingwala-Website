
     - Project Name - Paintingwala Website Testing
     
     1. Introduction
           This test plan is created to check whether the Paintingwala website is working properly. It focuses on testing features like navigation, contact form, UI, mobile compatibility, and basic security.

    2.  Scope
     We will test the following:
      - Website functionality
      - UI and layout
      - Contact form
      - Navigation menu
      - Mobile responsiveness
      - Basic security checks
  
    3. Scope – What We Will Not Test
      - Backend functionality
      - Database testing
      - Performance/load testing
      - Payment gateway (if not available)

    4. Devices for Testing
      - Laptop/Desktop (Chrome browser)
      - Mobile device (Redmi Note 11T 5G)

    4. Testing Types
    1. Function Check
          - Verify all features are working properly
    2. UI Check
          Verify layout, alignment, and design
    3. Security Check
          - Verify no malicious scripts are accepted
          - Session/logout works properly
    4. Mobile Compatibility Check
          - Verify website works on mobile devices
          - Check responsiveness and layout

    5. Test Scenarios
      - Contact form submission works
      - Error shown for invalid inputs
      - All buttons and links are clickable
      - Navigation menu works properly
      - Images load correctly
      - Website loads on mobile devices
      - No horizontal scrolling issue
      - Website uses HTTPS
      - Form does not accept malicious scripts
